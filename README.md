# Builderator
__Tools to make CI Packer builds awesome__

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'builderator'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install builderator

## Usage

Run `bundle exec thor build help`
